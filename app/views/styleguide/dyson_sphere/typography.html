Fonts

Primary Font: Quatro Slab

Variable name: $header-font

Used for headlines, sub headlines, short body copy, hyperlinks and CTA buttons

Header Font Weights

We have 3 different font weights we use: `normal`, `bold`, and `$medium` (which maps to the CSS value of `500`). Use any of these three values when setting font weight for a header font (and preferably not any other values besides these three options.)

Title Lines

If you want a particular header to have nice lines on either side of it, include the mixin title-line(color) to the styles you're defining for that header. Example:

@include title-line($la-grey);
